# Imogen's Vision — Website Build Plan

## Tool Workarounds
- **Reading files**: Use `launch-process` with `cat` if `view` tool fails
- **Creating/rewriting files**: Use `remove-files` + `save-file` if `str-replace-editor` fails
- **Task tracker**: Skip Augment tracker — communicate progress directly in chat
- **On failure**: Retry immediately with alternative tool, don't stop to discuss

---

## Phase 1 — Foundation
Design system, base CSS, shared components.
- Colour palette as CSS variables (sage greens, warm cream, charcoal)
- Typography (Inter / modern sans-serif, sizes, weights)
- Shared navigation (responsive, mobile hamburger, text wordmark)
- Shared footer (disclaimer, social placeholders, privacy link)
- Base component styles (buttons, cards, containers, form inputs)

## Phase 2 — Home Page
- Hero section (headline, subtitle, CTA, placeholder image)
- Introduction (short warm intro to Imogen)
- What she helps with (3-4 cards: coaching, confidence, habits, accountability)
- Why work with Imogen (differentiators)
- Motivational quote block
- Call to action section

## Phase 3 — About Imogen
- Her story (warm personal narrative)
- Values and approach (progress not perfection, realistic steps)
- Her message (standout quote block)
- Placeholder portrait photo alongside text

## Phase 4 — Work With Imogen
- Services overview (coaching, confidence, habits, accountability, lifestyle change)
- How to engage (consultation, individual coaching, ongoing packages, bespoke/clinic)
- CTA to contact page

## Phase 5 — FAQ
- Written FAQ content (6-8 questions)
- Accordion/expandable layout

## Phase 6 — Contact
- Simple 3-field form (name, email, message)
- Contact info and social coming-soon placeholders

## Phase 7 — Privacy Policy
- Basic UK GDPR privacy policy page

## Phase 8 — Final Polish
- Mobile responsive testing
- Accessibility audit
- Cross-page consistency
- Browser testing

11. Video: None yet - design without, leave space
